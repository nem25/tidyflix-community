# TidyFlix — Community Issue/Feature Tracker

This repository contains no source code. It exists as a public space for bug
reports, feature requests, and general feedback for **TidyFlix** — a terminal
UI for renaming and organising TV show and movie files.

---

## 📦 Install TidyFlix
```bash
npm install -g tidyflix
```

Then run:
```bash
tidyflix
```

---

## 🐛 Reporting a Bug

Before opening an issue, please check whether one already exists for your
problem.

When filing a bug report, include:

- **TidyFlix version** — run `tidyflix --version`
- **Node.js version** — run `node --version`
- **Operating system** and terminal emulator
- Steps to reproduce the problem
- What you expected to happen vs. what actually happened

---

## 💡 Requesting a Feature

Feature requests are welcome. Describe the use case you're trying to solve
rather than just the solution — it helps prioritise what's actually useful.
